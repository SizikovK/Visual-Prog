# Задания для знакомства с typescript

1. Создать интерфейс **User**
2. Создать интерфейс **Book**
3. Написать функцию для вычисления площади фигуры
4. Создать тип **Status**
5. Объявить тип функции **StringFormatter** для двух фукнций:
    - Делает первую букву заглавной
    - Обрезает пробелы по краям строки
       
6. Написать фукнцию, которая возвращает первый элемент массива
7. Создать интерфейс HasId и функцию которая ищет в массиве объект с заданным id

Так же покрыл тестами всю программу (по крайней мере то что надо было)

### Запуск:
```

npm run dev       <--- Запуск программы
```
